### Installation
#### Recommended : Install virtualenv
```bash
$ pip install virtualenv
$ virtualenv venv
```
#### Install dependencies
```bash
$ pip install -r requirements.txt
```

### Running Flask App
To run the flask app
```bash
$ export FLASK_APP=app
$ export FLASK_ENV=development
$ flask run
```
